Gestión de Estudiantes

Descripción

Estás desarrollando un programa para gestionar la información de estudiantes en una escuela. Cada estudiante tiene la siguiente información:

  • Nombre
  • Edad
  • Calificaciones

Tu tarea es:

Crear un arreglo de objetos para almacenar la información de al menos 5 estudiantes. Implementar las siguientes funciones:

  • a-Una función que calcule el promedio de calificaciones de cada estudiante y lo muestre junto con su nombre.
  • b-Una función que determine qué estudiantes tienen un promedio mayor o igual a 7.
  • c-Una función que permita buscar un estudiante por nombre y mostrar toda su información.

Requerimientos

  • Usa arreglos y objetos.
  • Usa funciones para organizar el código.
  • Muestra resultados en la consola.

Ejemplo

Supongamos que tienes el siguiente arreglo de estudiantes:

const estudiantes = [ { nombre: "Ana", edad: 18, calificaciones: [8, 9, 7] }, { nombre: "Luis", edad: 20, calificaciones: [5, 6, 4] }, { nombre: "María", edad: 19, calificaciones: [9, 9, 10] }, { nombre: "Carlos", edad: 21, calificaciones: [6, 7, 5] }, { nombre: "Sofía", edad: 18, calificaciones: [10, 10, 9] } ];

Con las funciones, deberías obtener resultados como:

Promedios

  • Ana: 8.0
  • Luis: 5.0
  • María: 9.3
  • Carlos: 6.0
  • Sofía: 9.6

Estudiantes con promedio mayor o igual a 7:

Ana, María, Sofía.

Información de búsqueda:

Si buscas a "María", debería mostrar:

{ nombre: "María", edad: 19, calificaciones: [9, 9, 10] }
